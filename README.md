# My Web Application Template

* Sinatra
* Livereload
* Automatic asset build
  * coffeescript
  * stylus

## Prerequisite

* ruby
* node.js
  * npm
  * grunt-cli

## Install

```
$ git clone https://github.com/choplin/my-webapp-templte
$ cd my-webapp-template
$ npm install
$ grunt init
```

## Run

```
$ bundle exec foreman start
```
